## Git Commit 메세지 분류

| Commit type          | Emoji                                            |
| -------------------- | ------------------------------------------------ |
| 최초 커밋 : init     | 🎉 `:tada:`                                       |
| 기능 추가  : feat    | 🆕 `:new:`                                        |
| 버스 픽스 : fix      | 🐛 `:bug:`                                        |
| 문서화 : docs        | 📝 `:memo:`                                       |
| 디자인 수정 : design | 🎨 `:art:`                                        |
| 코드개선 : refactor  | 🔨 : `:hammer:`                                   |
| Other                | [Be creative](http://www.emoji-cheat-sheet.com/) |



ex)

:new: feat : backend 게시글 등록 기능 추가
